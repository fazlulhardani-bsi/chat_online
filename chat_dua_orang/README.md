# Chat Online - Peer to Peer

Website chat online yang bisa digunakan untuk chat 2 orang dari kota berbeda, tanpa perlu setup server!

## Cara Pakai:

### 1. Deploy ke Hosting Gratis

#### Opsi A: Netlify (Paling Mudah - 1 Menit!)
1. Buka https://app.netlify.com/drop
2. Drag & drop file `index.html` ke halaman tersebut
3. Tunggu upload selesai
4. Dapat link (contoh: https://random-name.netlify.app)
5. Share link tersebut ke teman!

#### Opsi B: Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Jalankan: `vercel`
3. Ikuti instruksi
4. Dapat link deployment

#### Opsi C: GitHub Pages
1. Upload `index.html` ke repository GitHub
2. Aktifkan GitHub Pages di Settings
3. Akses via `https://username.github.io/repo-name`

### 2. Cara Chat dengan Teman:

**Orang Pertama (Pembuat Room):**
1. Buka link website
2. Klik "Buat Room Baru"
3. Akan muncul kode room (contoh: ABC123)
4. Klik kode room untuk copy
5. Share kode tersebut ke teman via WA/Telegram/dll

**Orang Kedua (Yang Gabung):**
1. Buka link website yang sama
2. Masukkan kode room yang diterima
3. Klik "Gabung ke Room"
4. Langsung terhubung dan bisa chat!

## Fitur:
- ✅ Peer-to-peer (langsung antar browser, tanpa server)
- ✅ Bisa diakses dari kota berbeda
- ✅ Tidak perlu install atau setup apapun
- ✅ Gratis selamanya
- ✅ Real-time messaging
- ✅ UI modern dan responsive
- ✅ Copy kode room dengan 1 klik

## Catatan:
- Koneksi peer-to-peer menggunakan PeerJS dengan server publik gratis
- Kedua orang harus online bersamaan untuk bisa chat
- Pesan tidak disimpan (hilang saat refresh)
- Untuk chat lebih dari 2 orang, perlu modifikasi kode
